# [Stock Screener Generator](#/use/screenerGenerator?id=stock-screener-generator)

> **IMPORTANT:** This page explains how to use the in-built Stock Screener Generator. If you want to learn all about the underlying function `SF_SCREEN()` then please see the [Stock Screener Function](#/functions/screener) page.

The Stock Screener Generator automatically builds the complex `SF_SCREEN()` functions for you with a super intuitive interface. Simply add the filters you want to the screen, select the metrics you want to see in the output apply any further options/formatting and then click **Generate** to enter the function into the currently active (highlighted) cell. The Stock Screener is available from the dropdown menu in Google Sheets, below the Function Generator.

![Stock Screener Location](https://cdn.sheetsfinance.com/public-site/img/docs/screener_sidebar_2.png)

Once you open the Stock Screener, you can begin to add filters from the dropdown menu at the top. The function display will show you what's going on under the hood as it builds the `SF_SCREEN()` function for you in real-time. You can toggle whether to display a metric with the visibility button on the right of each filter. If you decide you no longer want a filter you can remove it by clicking the top-right delete button on the filter. Below the filters are the various options, such as the order (which defaults to market cap descending), the limiting of results ad well as the removal of the header rows and reverse order (ascending).

![Stock Screener Generator](https://cdn.sheetsfinance.com/public-site/img/docs/screener_sidebar_1.png)
